
# To-do List

Uma aplicação web simples para gerenciamento de tarefas, desenvolvida com HTML, CSS e JavaScript puro. O objetivo é praticar manipulação do DOM, eventos e armazenamento local.

## Funcionalidades

- Adicionar novas tarefas
- Editar tarefas existentes
- Marcar tarefas como concluídas
- Remover tarefas
- Pesquisar tarefas por texto
- Filtrar tarefas (todas, feitas, a fazer)
- Salvar tarefas localmente (localStorage)

## Como usar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Utilize o campo de texto para adicionar tarefas.
4. Use os botões para marcar como feito, editar ou remover.
5. Pesquise ou filtre tarefas usando a barra de ferramentas.

## Estrutura do projeto

- `index.html`: Estrutura da página
- `css/styles.css`: Estilos visuais
- `js/scripts.js`: Lógica da aplicação
